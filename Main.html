What is Night Vision?

●        It is the ability to see in the dark or when there is low /no light. So the night vision is, in which we can see in the dark not as same as in bright light like in day but we can at least see.

●        There are some things which we should know before we going further like spectrum of light.










Spectrum of Light

●        There are mainly two types of light visible and non visible.

●        It is measured in some range called intensity range.




●        Visible light’s range starts from 380 nm (nanometer) to 700 nm.

●        Non visible light’s range includes all except visible light range.

 

   Classification of Night Vision:

  Biological Night Vision:

●        In biological night vision it depends on visible light, there is no equipment, basically it is within human’s retina. In human’s eye, beside retina there is one tissue called TAPETUM LUCIDUM which is very sensitive to the light.

 

   Night Vision Technology (Manmade)

 

●        A Night Vision Technology means, by which we can see in the dark or we can increase the ability to see in dark through any equipment, which is developed by us.

 

 Classification of Night Vision Technology

Night Vision Technology Broadly classified into 3 categories till now:

     1) Image Intensification:

●        Basically this is passive type of night vision technology. According to the spectral range it is based on visible light and because of it, it is passive night vision technology.

●        In this type whenever there is low light (it means visible light is present there but having very low intensity), it captures it in a device in which there is vacuum through the lens. After that the light passes through the photocathode and because of this the light particles convert into the electron. After all the electrons will passes through the Micro channel plate which multiplies the no of electron to make image more visible or bright. After all the electrons passes through the phosphor screen which converts the electrons into the light particles or we can say into an image and we can see through the other lens. 4.1.1  Active Illumination

●        Active illumination is a subtype of Image intensification. In active illumination it uses Near Infrared (NIR) or we know it as Infrared (IR) light. infrared has wavelength upto 1050 nm and that's why we can sometimes see it at certain angle.

●        as we discuss that IR has wavelength between 700 to 1000 nm and having this range so it is place right after the visible light. And CCD (Charge Coupled Device)cameras are sensitive to this light.

●        So whenever we use it,first it emits the IR light  through the illuminators (illuminators can be bulb type or LED type or Laser type)and then it receives the infrared light from the object because infrared light is placed after visible light. So it is obvious that we can clearly see in the night but it is monochrome.



Fig 4.3: Active Illumination

●     According to the fig. 4.1, whenever we use Active IR then, first we emits the IR through illuminator (Here illuminator is in the helmet) then it goes into the vacuum tube through the lens and then IR light converts into the electrons through photocathode and then it goes into the microchannel for multiplying the electrons and then we can see it after it passes through the phosphor layer and then we saw the image through another lens.

●     Nowadays active illumination is mostly used  in commercial, residential and in government security applications for surveillance at night. But it is not used in military because there are more risk while using it. If they used it then they may be caught by their enemies  

2) Thermal Imaging  

●     Thermal Imaging also uses the infrared but it has longer wavelength like 9,000 - 14,000 nm. and it uses uit as radiation long-range infrared  and after receiving that data it creates an image which is called thermograms.

●     Infrared radiation is emitted by most of all the object with some temperature. And all the object has different temperature so in thermograms we can easily differentiate the object.

●     There is IR film which also be used for measuring the temperature and creates the image but the range of it has to be in between 280 °C to 500 °C. It is very high temperature and it is not feasible in day to day life. That's why we use thermograms instead of IR film.



●     In real life it can be used in condition mapping which is used to check the faults in machineries and it also be used by military.

There are two types of this. Active and Passive

●     in Active thermal imaging we don’t need any source of IR and it use IR which are emitting through the object.

●     In passive thermal imaging we need a source of IR it can be anything.
